# Workstation

Interactive diagram visualization for workstation architecture.

## Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Runtime:** Bun
- **Language:** TypeScript
- **Deployment:** GitHub Pages

## Inspiration

This project is inspired by [akexorcist/workstation-diagram](https://github.com/akexorcist/workstation-diagram), which visualizes workstation setups in an interactive diagram format.

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Deployment

Built as a static site and deployed to GitHub Pages via GitHub Actions.
